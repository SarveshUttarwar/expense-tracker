This Expense Tracker is a smart personal finance management application designed to simplify the way users track and manage their expenses. The app combines traditional expense tracking with AI-powered features like voice-based expense entry and automatic bill scanning to provide a seamless user experience.

Features
🎤 Voice-Based Expense Entry

Users can add expenses using voice commands instead of manually entering details.

Example:

“Spent 500 rupees on groceries”
“Paid 1200 for electricity bill”

The app processes the command and automatically creates a transaction entry with the relevant details.

🧾 Bill Upload & Automatic Data Extraction

Users can upload images or PDFs of bills and receipts.

The app automatically:

Extracts important information from the bill
Identifies:
Amount
Vendor/Store name
Date
Category (if applicable)

The extracted information is then added directly to the transactions list.

💳 Transaction Management

All expenses are stored and displayed in the Transactions section.

Users can:

View all transactions
Track spending history
Monitor categorized expenses
Keep a centralized record of all financial activity
📊 Expense Analytics

The application provides detailed analytics and visual insights into spending habits.

Analytics include:

Category-wise spending breakdown
Monthly expense summaries
Spending trends
Total expenses overview

These insights help users better understand and manage their finances.

📈 Monthly Comparison

Users can compare their current month’s spending with previous months.

The comparison feature helps users:

Identify spending patterns
Analyze increases or decreases in expenses
Make better budgeting decisions
